# Quick Start Guide - Workspace Invitation

## For Workspace Owners (Sending Invitations)

1. **Navigate to your workspace**
   - Go to any workspace you own

2. **Open Invite Modal**
   - Look for "Invite Members" button in Sidebar
   - Click it

3. **Enter Email**
   - Type the email of the user you want to invite
   - Email must be registered in the system

4. **Send Invitation**
   - Click "Send Invitation" button
   - Wait for success message

## For Invited Users (Accepting Invitations)

1. **Check Notifications**
   - Look for bell icon (🔔) in header
   - Red badge shows unread count

2. **Open Notification Panel**
   - Click bell icon
   - See list of notifications

3. **Accept Invitation**
   - Find workspace invitation (📨 icon)
   - Click "Accept" button

4. **Access Workspace**
   - Go to Landing Page
   - Find workspace in "Your Workspaces" section
   - Click to enter

## Troubleshooting

**Error: "User not found"**
- Email must be registered first
- Ask user to sign up

**Error: "Already a member"**
- User is already in workspace
- No action needed

**Notification not appearing**
- Check WebSocket connection
- Refresh page
- Check browser console for errors
